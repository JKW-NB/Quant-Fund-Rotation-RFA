# 🚀 基于高阶统计矩与 RFA 双重集成的公募基金量化轮动系统
**Quantitative Fund Rotation System via High-Order Moments & RFA Ensemble**

本项目旨在跳出传统的“均值-方差”线性框架，在 A 股主动权益类公募基金的横截面上寻找 Alpha 超额收益。通过引入统计学高阶矩（偏度、峰度）重构因子库，并采用 Random Forest-AdaBoost (RFA) 双重集成算法捕捉尾部风险，构建了一个包含严苛实盘摩擦成本的月频轮动回测系统。

## 🌟 核心亮点 (Core Highlights)

- **严谨的数据基建**：自动化对接 AkShare 接口，精准下沉至 100 只主动偏股型基金。剔除被动 ETF 避免异质性污染，通过 `len(data)>800` 强过滤**生存偏差 (Survivorship Bias)**，并全程使用“累计净值”规避分红陷阱。
- **高阶统计特征工程**：除传统的动量与下行波动率外，深度提炼了反映收益非对称性与肥尾效应的**偏度 (Skewness)** 与 **峰度 (Kurtosis)** 因子，并执行日频 Z-score 截面标准化剥离系统性 Beta。
- **算法架构降维打击**：针对单体树模型集中持仓导致的“夏普比率衰减”，引入 RFA 双重集成架构。利用 AdaBoost 对错分样本的自适应指数惩罚，强迫底层随机森林极度敏锐地规避极易发生大幅回撤的“尾部高危基金”。
- **实盘级回测引擎**：从零编写向量化回测系统。严格设置 `shift(1)` 信号延迟防止未来数据穿越，采用月度调仓降频，并逐笔扣除 `0.15%` 的真实综合摩擦成本。

## 📊 绩效战报 (Out-of-Sample Performance)
